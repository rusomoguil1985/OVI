Desarrollo de Software

Colombia Digital (2014). ¿Que es programar? [Pagina Web] Recuperado de: https://www.youtube.com/embed/67C6IeTJ90s

Bienvenidos a este OVI enfocado a la introducción al desarrollo de software.

Para conocer que es desarrollar software, primero debemos saber ¿Qué es un Software? Pues bien, podríamos decir que el software es todo el sistema lógico de un computador, este sistema se encuentra programado previamente para que pueda así llevar a cabo la realización de tareas específicas, normalmente un software se crea como producto y alternativa al cumplimiento de una necesidad identificada, un software cumple unas funciones lógicas en interacción con el usuario y al tiempo con el hardware del equipo.

Hay software bastante conocidos y que podemos usar diariamente, como lo son los editores de texto (Word, Write), hojas de cálculo (Excel, spreadsheet), videojuegos, aplicaciones bancarias, navegadores de internet, redes sociales entre otras.

Ahora bien, ya que sabemos que es software, planteemos la pregunta ¿Qué es desarrollo de software? El software es creado por personas comúnmente conocidas como desarrolladores, los desarrolladores programan los software por medio de lenguajes de programación, allí definen las funciones de cada uno de los elementos que componen el software, ya sean Botones, formularios, acciones con mouse, combinaciones de Teclado, entre otras.

Esta combinación de funcionalidades da al funcionamiento correcto del sistema de información. Hay diferentes tipos de programación, podría decirse que en la actualidad la más popular es la programación orientada a objetos y eventos y hay diferentes lenguajes que se enfocan al desarrollo de aplicaciones por esta metodología, Pero el desarrollo de software es un proceso largo y complejo que se divide en diferentes etapas, metodologías y modelos de desarrollo, desde su planeación hasta su etapa de producción, así que vamos a conocer los aspectos que componen el desarrollo de software.
